######   I N T U I T I O N
    ### Problem 1784: Check if Binary String Has at Most One Segment of Ones
    **Logic:** Pattern Matching ("01" detection)
    **Complexity:** O(N) time, O(1) space
    
    **Intuition:**
    Since the string is guaranteed to start with '1', any additional segments of '1's 
    can only exist if there is a '0' followed by a '1' (a "01" substring). 
    If we detect this pattern, it means the ones are not contiguous.
<img width="665" height="368" alt="image" src="https://github.com/user-attachments/assets/f50623b7-8f2c-4c24-973c-6f50c642abd8" />

###### C O D E
```cpp
class Solution {
public:
    bool checkOnesSegment(string s) {
        // Since s[0] is always '1', we just need to check if "01" exists.
        for(int i = 1; i < s.length(); i++) {
            if(s[i-1] == '0' && s[i] == '1') {
                return false; 
            }
        }
        return true;
    }
};
